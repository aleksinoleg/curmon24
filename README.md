# uah-rate-bot

Моніторинг курсу USD/UAH: Приват24 (безготівковий) + міжбанк, зі сповіщеннями в Telegram.

**Фаза 1:** пасивний моніторинг і логування. Сповіщення коли:
- Приват24 змінив курс купівлі/продажу
- міжбанк зрушив на ≥ 0.15 грн (поріг — `ALERT_THRESHOLD` у workflow)

Дані пишуться в `data/rates.csv` — це основа для калібрування сигналу «продавати/чекати» у Фазі 2 (через 1–2 тижні збору даних).

## Налаштування (один раз, ~5 хвилин)

1. Створи **публічний** репозиторій на GitHub (публічний = безліміт хвилин Actions;
   приватний на free-плані має 2000 хв/міс — цього НЕ вистачить при полінгу кожні 15 хв).
2. Запуш ці файли в репозиторій.
3. У репозиторії: **Settings → Secrets and variables → Actions → New repository secret**:
   - `TELEGRAM_BOT_TOKEN` — токен від @BotFather
   - `TELEGRAM_CHAT_ID` — твій chat ID
4. Вкладка **Actions** → увімкни workflows → відкрий `rate-monitor` → **Run workflow**
   (перший ручний запуск, перевірити що все працює).
5. Напиши боту `/start` в Telegram (щоб він міг надсилати тобі повідомлення).

## ⚠️ Безпека

- Токен зберігається **тільки** в GitHub Secrets. Ніколи не комітьте його в код —
  репозиторій публічний.
- Якщо токен колись «світився» у чатах/логах — перевипустіть його через @BotFather
  (`/mybots` → бот → API Token → Revoke) і оновіть секрет.

## Локальний тест

```bash
pip install -r requirements.txt
python monitor.py --debug        # фетчить курси, друкує, нічого не надсилає
```

## Нюанси GitHub Actions

- Cron у GitHub — не точний: запуски бувають із затримкою 3–15 хв. Для нашої задачі це ок.
- Розклад у UTC. Влітку Київ = UTC+3, взимку UTC+2 — вікно в workflow взято з запасом.
- У публічних репо scheduled workflows вимикаються після 60 днів без активності,
  але бот сам комітить дані щодня, тож це не проблема.

## Структура

```
monitor.py                     — увесь код (фетч, стан, сповіщення)
.github/workflows/monitor.yml  — розклад полінгу
data/state.json                — останні значення (створюється автоматично)
data/rates.csv                 — журнал курсів (створюється автоматично)
```
